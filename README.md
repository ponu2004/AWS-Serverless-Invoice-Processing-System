AWS Invoice Processing System

## Project Overview
This is a serverless invoice processing web application built using AWS services.  
Users can log in, upload invoices, and process them automatically in the cloud.

---

## Technologies Used

- AWS Cognito (Authentication)
- Amazon S3 (Storage)
- AWS Lambda (Processing)
- Amazon Textract (Text extraction)
- AWS Amplify (Hosting)
- HTML, JavaScript

---

##  Project Workflow

1. User logs in using Cognito Hosted UI
2. User uploads invoice
3. File stored in S3 bucket
4. Lambda triggers
5. Textract extracts invoice data
6. Results processed automatically
